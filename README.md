# Task Manager API

Backend API for the Task Manager application.

Built with **Django REST Framework** and secured with **JWT authentication**.

## Features

* User authentication
* Secure API endpoints
* CRUD operations for tasks
* User-owned task data
* Token-based security

## API Routes

Authentication
/api/auth/register
/api/auth/login
/api/auth/token/refresh

Todos
/api/todos/

## Tech Stack

* Django
* Django REST Framework
* SimpleJWT
* PostgreSQL (production)

## Deployment

Hosted on Render.
